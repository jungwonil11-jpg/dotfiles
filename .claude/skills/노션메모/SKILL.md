# 노션메모 — 직전 대화 내용을 노션 학습 노트에 추가

직전 대화에서 설명한 개념·내용을 노션에 학습 노트 스타일로 추가한다.

## 작성 스타일
- **음슴체** 사용 (예: "~임", "~함", "~됨")
- **비유** 적극 활용 — 낯선 개념은 익숙한 것에 빗대어 설명
- **표** 사용 — 비교, 속성 정리에 적합할 때
- **Callout 블록** 사용 — 핵심 요점, 주의사항, 팁 강조에 활용
- 코드 예제는 코드 블록으로 포함

## 실행 순서

### 1. 노션 검색
직전 대화의 핵심 키워드로 노션을 검색해 이미 해당 내용이 있는지 확인.
- 이미 있으면: 추가하지 말고 해당 페이지 링크만 사용자에게 알려줄 것.
- 없으면: 2단계로 진행.

### 2. 추가할 위치 파악
노션에서 적합한 상위 페이지(예: Python 학습 노트, day01 등)를 찾아 확인.
- 적합한 페이지가 없으면 사용자에게 어디에 추가할지 물어볼 것.

### 3. 학습 노트 작성 및 추가
직전 대화 내용을 위 스타일에 맞게 정리하고 노션 페이지에 추가.

### 4. 완료 알림
추가된 노션 페이지 링크를 사용자에게 알려줄 것.
